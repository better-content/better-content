# Remaining Disabled Mods: Easy-Win Review

Context: this is after the requested re-enable profile. These mods remain disabled outside the active repo profile. The requested re-enable set plus required dependencies profiled at `12.58 GiB` RSS at loaded menu, so the practical question is what should stay out unless it buys a very clear progression payoff.

These are low-signal or mostly aesthetic/model-surface mods. Re-adding them is likely to grow the block/item/model atlas without solving a progression need.

The requested set already re-added Aether, Blue Skies, Deeper and Darker, Fallout Wastelands, Finley, Lost Cities, Twilight Forest, and Undergarden. Adding more dimension/worldgen surface is not compatible with the 8 GiB target unless another branch is cut.

The requested set did not re-add trains or Little Logistics. Given the requested set is already `12.58 GiB`, these are not easy to afford now.

## Keep Disabled: EMI Addons

Base EMI was re-enabled. Keep addons out until the core memory budget is acceptable.

- EMI Enchanting
- EMI Loot
- EMI Ores
- EMI Trades

## Maybe Later, But Not Easy Wins

These could become useful if tied to a strong progression role, but they are not cheap enough to add casually under the current budget.

- Create: Steam 'n' Rails: design-core logistics, but too expensive while the requested content set is active.
- Little Logistics: thematically aligned local logistics, but redundant if trains return.
- The Bumblezone: distinctive dimension, but another dimension branch is hard to justify under RAM pressure.
- Hex Casting: strong post-AE2/programmatic magic candidate, but not as a casual side magic re-add.
- Ars Energistique: useful only if AE2/source integration becomes a late hybrid branch.
