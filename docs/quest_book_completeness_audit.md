## Resolved Gaps

- Dependency graph now exists and is validated.
- Starting Out now has shelter, food/water, Nether prep, netherrack grout, meltery, and terminal exits.
- TCon and Create chapters are dependency-linked.
- Food/body/sacrifice bridge is represented by `food_body` and `death`.
- Wares and villager economy are represented as crafting/economy systems.
- Create III logistics and OC2R route logic are represented.
- AE2 is described as local site intelligence.
- Post-AE2 has explicit branches.
- Synthesis has Acid Vat and Chemlib plate milestones.

## Remaining Manual Checks

These cannot be proven by static SNBT validation:

- FTB Quests UI rendering of generated `description` arrays.
- Whether `onboarding_complete` should be a real FTB stage gate once the onboarding mod exposes/mirrors that state.
- Whether players find the chapter order readable in-game.
- Whether quest rewards feel balanced against actual obelisk/dimension coin income.

## Next Quest Work

The next quest-book work should be playtest-driven, not broad skeleton generation:

- Tune quest text and coordinates after viewing the book in-game.
- Add explicit onboarding stage visibility when the final stage/event hook is confirmed.
- Add deeper dimension/boss/adventure subchapters after the obelisk/dimension reward hierarchy is finalized.
- Add optional per-mod detail quests only where players need teaching, not for every item.
